# Selenium-WebDriver Boilerplate
    
Selenium-WebDriver end-to-end test automation boilerplate, Using [Commonjs](https://requirejs.org/docs/commonjs.html), [Cucumber](https://github.com/cucumber/cucumber-js), [Chai](https://www.chaijs.com) and Assert.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/selenium-webdriver-commonjs-cucumber-chai-assert.git
    	2. cd selenium-webdriver-commonjs-cucumber-chai-assert
    	3. npm install
    	4. npm run test
        
    
For more boilerplate click [here](https://github.com/e2e-boilerplate/utils/blob/master/docs/implemented.md)
